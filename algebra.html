Radicals & Complex Numbers Q1
Q1Multiple Choice

Which of the following is the simplified form of √75?

  • A 3√5
  • B 5√3
  • C 15√5
  • D 25√3
✅ B — 5√3. Factor 75 into a perfect square times something: 75 = 25 × 3. So √75 = √(25 × 3) = √25 · √3 = 5√3. The key step is identifying the largest perfect square factor. 25 is the largest perfect square that divides 75, giving a fully simplified result.
Q2Multiple Choice

What is the rationalized form of 6 / √3?

  • A 2
  • B 6√3
  • C 2√3
  • D 3√2
✅ C — 2√3. Multiply numerator and denominator by √3: (6/√3) · (√3/√3) = 6√3 / 3 = 2√3. Rationalization removes the radical from the denominator by multiplying by a form of 1. Always simplify the result — here 6/3 = 2.
Q3Multiple Choice

Simplify: (2 + √5)(2 − √5)

  • A −1
  • B 4 − √5
  • C 9
  • D 4 + √5
✅ A — −1. This is a conjugate pair: (a + b)(a − b) = a² − b². So (2 + √5)(2 − √5) = 2² − (√5)² = 4 − 5 = −1. Conjugate products always yield a rational number, which is exactly why we use conjugates to rationalize binomial denominators.
Q4Multiple Choice

Solve: √(x + 3) = 4. Which value of x is the solution?

  • A x = 1
  • B x = √7
  • C x = 19
  • D x = 13
✅ D — x = 13. The radical is already isolated. Square both sides: (√(x+3))² = 4² → x + 3 = 16. Squaring gives the LEOV — a linear equation in one variable: x + 3 = 16. Solve: x = 13. Always check in the original: √(13 + 3) = √16 = 4 ✓. No extraneous solution here.
Q5Multiple Choice

What is i³ equal to?

  • A 1
  • B −i
  • C i
  • D −1
✅ B — −i. The powers of i cycle with period 4: i¹ = i, i² = −1, i³ = −i, i⁴ = 1, then repeats. So i³ = i² · i = (−1)(i) = −i. To find iⁿ quickly, divide n by 4: remainder 0 → 1, remainder 1 → i, remainder 2 → −1, remainder 3 → −i.

Functions & the Coordinate Plane Q2
Q6Multiple Choice

Which of the following relations is a function?

  • A {(1,2), (1,3), (2,4)}
  • B {(3,1), (3,2), (4,5)}
  • C {(1,5), (2,5), (3,5)}
  • D {(2,4), (2,−4), (3,9)}
✅ C. A function requires every x-value to map to exactly one y-value. In A and B, x = 1 and x = 3 each appear twice with different y-values — not a function. In D, x = 2 maps to both 4 and −4 — not a function. In C, each x (1, 2, 3) maps to exactly one y (5 each). Having the same output for different inputs is perfectly fine — that just means it's not one-to-one.
Q7Multiple Choice

If f(x) = 2x² − 3x + 1, what is f(−1)?

  • A 6
  • B −4
  • C 0
  • D 4
✅ A — 6. Substitute x = −1: f(−1) = 2(−1)² − 3(−1) + 1 = 2(1) + 3 + 1 = 6. Be careful: (−1)² = +1, not −1. A common error is applying the negative sign to the exponent before squaring. Always square first, then apply the coefficient.
Q8Multiple Choice

What is the distance between (1, 2) and (4, 6)?

  • A √7
  • B 3
  • C √21
  • D 5
✅ D — 5. d = √((4−1)² + (6−2)²) = √(9 + 16) = √25 = 5. This is actually a 3-4-5 right triangle scaled naturally. The distance formula is the Pythagorean theorem applied to the coordinate plane — the horizontal leg is 3, the vertical leg is 4, so the hypotenuse is 5.

Q9Multiple Choice

What is the slope of the line passing through (−2, 5) and (4, −1)?

  • A 1
  • B −1
  • C 2
  • D −2
✅ B — −1. m = (y₂ − y₁)/(x₂ − x₁) = (−1 − 5)/(4 − (−2)) = −6/6 = −1. A negative slope means the line falls from left to right. Note: the order of the points doesn't matter as long as you're consistent — subtracting in the same order in both numerator and denominator.
Q10Multiple Choice

What is the value of the discriminant of 2x² − 4x + 2 = 0, and how many real roots does it have?

  • A Discriminant = 8; two distinct real roots
  • B Discriminant = −16; no real roots
  • C Discriminant = 0; one repeated real root
  • D Discriminant = 4; two distinct real roots
✅ C — Discriminant = 0; one repeated real root. With a=2, b=−4, c=2: D = b² − 4ac = (−4)² − 4(2)(2) = 16 − 16 = 0. When D = 0, there is exactly one real root (a double root). The parabola touches the x-axis at exactly one point — its vertex. In this case x = −b/2a = 4/4 = 1.

Radicals & Complex Numbers — Problem Solving Q1
PS1Short Answer

Simplify completely: √48

Write your answer using the √ symbol, e.g. 5√2

✅ 4√3. Factor: 48 = 16 × 3. So √48 = √16 · √3 = 4√3. Always look for the largest perfect square factor — using 4 × 12 first would give 2√12, which needs to be simplified again to 4√3.
PS2Short Answer

Simplify: 3√5 + 2√20

✅ 7√5. First simplify √20 = √(4·5) = 2√5. Then: 3√5 + 2(2√5) = 3√5 + 4√5 = 7√5. Like radicals can only be combined after they share the same radicand — here both become √5 after simplification.
PS3Short Answer

Expand and simplify: (√2 + 3)(√2 − 1)

Combine all like terms in your final answer.

✅ 2√2 + 2. FOIL (√2 + 3)(√2 − 1): √2·√2 − √2·1 + 3·√2 − 3·1 = 2 − √2 + 3√2 − 3 = 2√2 − 1. Wait — that's 2 − 1 + (−1 + 3)√2 = 1 + 2√2. Let me redo carefully: (√2)(√2) = 2, (√2)(−1) = −√2, (3)(√2) = 3√2, (3)(−1) = −3. Sum: 2 − √2 + 3√2 − 3 = (2−3) + (3−1)√2 = −1 + 2√2 = 2√2 − 1. ✅ Answer: 2√2 − 1.
PS4Short Answer

Solve for x: √(2x − 3) = 3

✅ x = 6. The radical is already isolated. Square both sides: 2x − 3 = 9 — this is the LEOV, a linear equation in one variable. Solve: 2x = 12, x = 6. Check: √(12 − 3) = √9 = 3 ✓.
PS5Short Answer

Multiply: (3 + 2i)(4 − i)

Write in a + bi form.

✅ 14 + 5i. FOIL: 3·4 + 3·(−i) + 2i·4 + 2i·(−i) = 12 − 3i + 8i − 2i². Since i² = −1: = 12 − 3i + 8i − 2(−1) = 12 + 2 + 5i = 14 + 5i.

Functions — Problem Solving Q2
PS6Short Answer

Given f(x) = x² + 2 and g(x) = x − 1, find (f ∘ g)(3).

✅ 6. (f ∘ g)(3) = f(g(3)). First g(3) = 3 − 1 = 2. Then f(2) = 2² + 2 = 6. Always evaluate the inner function first.
PS7Equation Form

Write the equation of the line passing through (0, 1) and (2, 5) in slope-intercept form.

✅ y = 2x + 1. Slope: m = (5 − 1)/(2 − 0) = 4/2 = 2. Since (0,1) is the y-intercept, b = 1. So y = 2x + 1.
PS8Short Answer

Solve: √(3x − 2) = 5

Isolate the radical, square both sides to get the LEOV, then solve and check.

✅ x = 9. The radical is already isolated. Square both sides: 3x − 2 = 25. This is the LEOV — a linear equation in one variable. Solve: 3x = 27 → x = 9. Check: √(27 − 2) = √25 = 5 ✓. No extraneous solution.

Linear Functions & Systems — Problem Solving Q3
PS9Graphing

Which graph correctly represents y = −2x + 3?

Think: slope = −2 (falls left to right), y-intercept = (0, 3).

  • A A line rising steeply, crossing y-axis at −3
  • B A horizontal line at y = 3
  • C A line falling left to right, crossing y-axis at 3 and x-axis at 1.5
  • D A line falling left to right, crossing y-axis at −2
✅ C. y = −2x + 3 has slope m = −2 (negative → falls from left to right, steep) and y-intercept b = 3 (crosses y-axis at 3). To find the x-intercept: set y = 0 → 0 = −2x + 3 → x = 1.5. So the line crosses (0, 3) and (1.5, 0), and falls steeply — matching option C.
PS10Equation Form

A line has equation y = −2x + 1. Write the equation of a line perpendicular to it that passes through (0, 4).

✅ y = ½x + 4. The slope of the original line is −2. A perpendicular slope is the negative reciprocal: −1/(−2) = ½. Since the line passes through (0, 4), the y-intercept is 4. So y = ½x + 4. Verify: (−2)(½) = −1 ✓ — slopes are negative reciprocals.
PS11Short Answer

Solve the system: y = x + 1 and y = 2x − 1. Write the solution as an ordered pair.

✅ (2, 3). Set equal: x + 1 = 2x − 1 → x = 2. Then y = 2 + 1 = 3. Check in both: 3 = 2 + 1 ✓ and 3 = 2(2) − 1 ✓. The solution is the point where the two lines intersect.
PS12Graphing

A system of two linear equations is graphed. The two lines are parallel and have different y-intercepts. How many solutions does the system have?

  • A Exactly one — they must intersect somewhere
  • B None — parallel lines never intersect
  • C Infinitely many — they share all points
  • D Two — one for each line
✅ B — No solution. Parallel lines (same slope, different y-intercepts) never intersect, so the system is inconsistent and has no solution. When you solve such a system algebraically, you get a false statement like 0 = 5, which signals no solution. If the lines were identical (same slope, same y-intercept), there would be infinitely many solutions.

Quadratic Functions — Problem Solving Q4
PS13Short Answer

Write the vertex of the parabola: f(x) = 2(x − 3)² − 4

✅ (3, −4). Vertex form is f(x) = a(x − h)² + k, so the vertex is (h, k) = (3, −4). Note: the h value is positive 3 because the form is (x − 3), not (x + 3). Since a = 2 > 0, the parabola opens upward and (3, −4) is the minimum point.
PS14Short Answer

Solve by factoring: x² + 3x − 10 = 0

List both solutions.

✅ x = 2 and x = −5. Find two numbers that multiply to −10 and add to 3: those are 5 and −2. So x² + 3x − 10 = (x + 5)(x − 2) = 0. By the zero-product property: x + 5 = 0 → x = −5, or x − 2 = 0 → x = 2. Check: (2)² + 3(2) − 10 = 4 + 6 − 10 = 0 ✓
PS15Graphing

For f(x) = −x² + 4x − 3, which of the following best describes the graph?

  • A Opens upward, vertex at (2, 1), x-intercepts at x = 1 and x = 3
  • B Opens upward, no x-intercepts, vertex below the x-axis
  • C Opens downward, vertex at (−2, 1), crosses x-axis at x = −1 and x = −3
  • D Opens downward, vertex at (2, 1), x-intercepts at x = 1 and x = 3
✅ D. a = −1 < 0 → opens downward. Axis of symmetry: x = −b/2a = −4/−2 = 2. Vertex y-value: f(2) = −4 + 8 − 3 = 1. So vertex is (2, 1). To find x-intercepts: −x² + 4x − 3 = 0 → x² − 4x + 3 = 0 → (x−1)(x−3) = 0 → x = 1 and x = 3. The parabola opens down, peaks at (2,1), and crosses x = 1 and x = 3.

How the Variables Work
a — shape & direction
Controls how wide or narrow the parabola is, and which way it opens.
• a > 0 → opens upward (U-shape) — has a minimum
• a < 0 → opens downward (∩-shape) — has a maximum
• |a| > 1 → narrower (steeper sides)
• 0 < |a| < 1 → wider (flatter)

h — horizontal shift
Moves the whole parabola left or right. The vertex x-coordinate.
• h > 0 → shifts right
• h < 0 → shifts left
⚠ In (x − h)², the sign flips: (x−3) means h=+3, shifting right.

k — vertical shift
Moves the whole parabola up or down. The vertex y-coordinate.
• k > 0 → shifts up
• k < 0 → shifts down

The vertex is always at (h, k) — the turning point of the parabola.

Slope-Intercept Form y = mx + b
Slope Formula m = (y₂ − y₁) / (x₂ − x₁) = rise / run
How the Variables Work
m — slope (steepness & direction)
How much y changes for every 1 unit you move right.
• m > 0 → line rises left to right (positive slope)
• m < 0 → line falls left to right (negative slope)
• m = 0 → perfectly horizontal line (no rise)
• |m| > 1 → steeper line (rises quickly)
• 0 < |m| < 1 → shallower line (rises slowly)
• undefined → vertical line (x = constant) — slope doesn't exist because division by zero: run = 0

b — y-intercept
Where the line crosses the y-axis (when x = 0).
• b > 0 → crosses y-axis above origin
• b < 0 → crosses y-axis below origin
• b = 0 → line passes through the origin

The equation y = mx + b lets you instantly read the slope and y-intercept directly from the formula.
